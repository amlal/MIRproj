This folder contains audio output examples.

ChromaOnly: 	      A file with samples that have chromatically increasing chroma

SpecCent: 	      A file with samples that have spectral centroid that 
increases to match a chromatic scale of sine waves

unweightedStravinsky: A file with an unweighted distance vector that attempts to recreate the first minute of the second movement of the Rite of Spring.

unweightedDylan:      A file with an unweighted distance vector that attempts to recreate the first minute of Bob Dylan's Tangled up in Blue with samples from the rest of the Blood on the Tracks album.
